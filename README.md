# Learning Style Detector 

Live Demo:
https://learning-style-detector.vercel.app/

An interactive Python-based Learning Style Detection System that identifies users’ preferred learning styles through quizzes, simulations, and personalized educational games.

---

##  Project Overview

This project analyzes how users prefer to learn by asking behavioral and educational preference questions based on learning style dimensions.

The system detects learning preferences such as:
- Active / Reflective
- Sensing / Intuitive
- Visual / Verbal
- Sequential / Global

The project evolved through multiple versions, starting from a simple console quiz to a more advanced personalized learning experience with browser-based educational games.

#  Versions Included

---

##  Version 1 — Basic Learning Style Detector

A simple interactive console-based simulation where users answer scenario-based questions to determine their learning style profile.

### Features
- Basic learning style analysis
- Scenario-based questions
- Console interaction

---

##  Version 2 — Quiz Game Version

An improved quiz-style detector with validation handling and balanced result detection.

### Features
- Multiple-choice quiz
- Input validation
- Balanced style detection
- Cleaner interaction system

---

##  Version 3 — Advanced Learning Style System

A more advanced version with:
- Multiple questions per dimension
- Personalized result generation
- Report exporting
- Personalized educational game launching

### Features
- Detailed learning analysis
- TXT report generation
- Replay system
- Personalized recommendations

---

##  Version 4 — Educational Browser Games Integration

This version connects each detected learning style with an online educational game or activity opened automatically in the browser.

### Learning Styles & Activities

| Learning Style | Activity Type |
|----------------|---------------|
| Active | Interactive learning games |
| Reflective | Logic & thinking games |
| Sensing | Practical educational activities |
| Intuitive | Pattern & strategy games |
| Visual | Visual creativity games |
| Verbal | Word & language games |
| Sequential | Step-by-step puzzles |
| Global | Big-picture quiz games |
